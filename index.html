About Me

I write code that ships, not just runs.

I work across the full stack — React frontends, REST APIs, Django backends — and deep into computer vision and LLM systems on the side. Real-time chat over WebSockets, object detection helping people navigate the world, storefronts running on Azure — these are deployed, used, and on GitHub.

I got into CV because I wanted to build something that mattered outside a browser. EyeSpy started as a question: what if a camera could be a person's eyes? That question turned into YOLOv8, OpenCV, and audio feedback in real time. That's the kind of problem I'm drawn to.

I pick up new tech fast, break things, fix them, and ship. That's the loop.
